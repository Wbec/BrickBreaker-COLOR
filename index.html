BRICK-BREAKER C O L O R
Will and Mykal's Brick Breaker

This is a project for Computer Science 130 offered at UVIC. It was created using:

Javascript: for game logic
JQuery: for HTML & CSS manipulation
JSON: for level storage and loading as DOM
HTML: for baisc markup
CSS: for styling and animations

Most of the project was completed with knowledge acquired within the course but there were a few resources that We used as research tools / references. Most notably, the use of W3Schools, MDN or Mozilla Developer Network StackOverflow (more detail in the code's comments) and our teacher's example were invaluable in the construction of this project.

Additionally, all of the projects source code is available on GitHub with documentation on how to implement the game into your own websites / create your own variant

Current Version: 1.0.1
